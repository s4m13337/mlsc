# Machine Learning & Scientific Computation

This repository contains research notes and experimental works related to Machine Learning and Scientific Computation. The programs are written with Python and Wolfram Mathematica.

## Prerequisites
After cloning this repository, it is recommended to create a virtual environment with:
```
cd mlsc
python3 -m venv .
. bin/activate
```

After the virtual environment is activated, install the required packages with
```
pip install -r requirements.txt
```
